# Dashboard screenshots

Add your screenshots to this folder using **exactly these filenames** so they show
up in the main README:

| Filename | What to capture |
|----------|-----------------|
| `overview.png` | The Overview page — stat tiles, jobs-by-status, throughput chart |
| `queues.png` | The Queues page — the queue table with live counts |
| `job-detail.png` | A job's detail page — execution attempts + logs |
| `workers.png` | The Workers page — the fleet with live/alive status |

Tip: start the stack with `docker compose up -d --build`, open
`http://localhost:5173` (login `demo@example.com` / `password123`), then capture
each page.
